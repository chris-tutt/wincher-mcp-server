# Usage Examples

50+ example prompts for using the Wincher MCP Server with Claude Desktop.

## Getting Started

### Basic Queries
```
"Show me all my tracked websites in Wincher"
"List all keywords we're tracking for [your-domain]"
"What's the status of my Wincher account?"
"How many keywords are we tracking total?"
```

## Tips for Better Queries

### Be Specific
❌ "Show me keywords"  
✅ "Show me keywords ranking 4-10 with search volume over 500"

### Use Context
❌ "How are we doing?"  
✅ "How are our SIEM-related keywords performing compared to last month?"

### Combine Tools
```
"Show me competitor rankings, then check the SERP for keywords where they beat us"
"List keyword groups, then show me detailed metrics for the worst performing group"
```

### Ask Follow-ups
```
You: "Show me our competitor comparison"
Claude: [shows data]
You: "Now show me the SERP for keywords where Splunk ranks #1 and we're outside top 10"
```
